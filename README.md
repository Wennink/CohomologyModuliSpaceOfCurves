# Cohomology of moduli spaces of curves.

We list computation results for the paper <insert_reference>.

We calculate the Sn-equivariant Hodge-Serre polynomial of moduli spaces of smooth curves, curves with rational tail, and curves of compact type.
We display these results using cohomology with compact support.

These results were computed using Deligne's weight spectral sequence.
Of particular note are the two new results for smooth curves: M5 and M3,3.
For these two spaces we used the pullback spectral sequence to calculate the weights <= 4 and the pushforward spectral sequence for the rest.

The correctness of these results depends on the verification of the Pixton conjecture for the tautological rings appearing on the first page of the spectral sequence.

If we calculate the tautological ring in the process of doing our calculations, then we check this against the known Betti numbers.
In the cases where we use H*(Γ,v), we have to verify the conjecture separately for the corresponding (Mg,n)^(S2^j).
For genus 0 and 1 the cohomology is fully known so there is nothing to check.
For (M2,6)^(S2^3), codimension <=4 and (M3,4)^(S2^2) codimension 5 we verify the conjecture separately.
